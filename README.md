Task tracker
============================

Использовался:
- react
- react-router
- nodejs
- express
- webpack
- mongo

------------

Возможности
------------
-Создание задач
-Редактирование
-Фильтрация задач


Установка
------------

1. npm i - установка зависимостей
2. mongo scripts/init.mongo.js - инициализация бд
3. npm run dev-all - запуск сервера
